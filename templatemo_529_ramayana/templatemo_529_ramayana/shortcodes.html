Manuel d installation

EASY COS est une application de bureau gratuite pour gestion de prêts electromenagers et sociaux, developpée par Notre Equipe dans le cadre du Projet pluridisciplinaire de la 2éme année classe préparatoire .Cette section a pour but de vous aidez a installler notre produit .

1.Vérifiez Vos Ressources :


Avant de commencer l'installation de l application , veuillez que vous avez tous les dossiers listés ci-dessous :

Dossiers Description Taille
Client.cache fichier Cache 20 KB
Prerequistes Contient les fichiers nécessaires pour le fonctionnement de l application 200 Mo
Setup Files Contient l'executable 38 KB
Easy Cos logo de l'application 3 KB

Si vous ne disposez pas de tout les fichiers veuillez cliquez ici pour les telecharger

2.Démarrer l'installation :

ouvrez le fichier "Setup files" et double-cliquez sur "Client" :



la fenetre d installation ci-dessous apparait , cliquez sur "Next" pour poursuivre ou sur "Delete" pour Annuler



ici l executable choisie par defaut "C:\ProgramFile(x86)" comme répertoire d installation, vous pouvez notamment le changer en cliquant sur "Browse" , dans le cas contraire poursuivez l installation en cliquant sur "Next"



Cliquez sur le boutton "install" Pour commencer l'installation



aprés que l installation est terminée , il vous suffit de cliquez sur "finish" et vous disposez maintenant de "Easy Cos" avec un raccourci qui se crée automatiquement sur votre bureau
